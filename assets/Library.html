Library

  1. Home / Academic
  2. Library

The library is holding more than 9800 books inclusive of text books and reference materials. Library subscribes to Newspapers, 6 Journals and E-resources. Students and faculty members can access the e-resources from anywhere using their library credentials. The library is automated using an Integrated Library Management System (ILMS) with OPAC search facility. It is user-friendly and supports all in-house operations of the library such as acquisition, cataloguing, circulation, etc. Library has access to the following legal and research databases. The library also provides remote access to subscribed resources to the authorized users. The library organizes orientation and training programmes to enable users to make effective use of resources and services. The library staff is always available to provide necessary assistance to the users. The reading room is spacious and fully air-conditioned.
